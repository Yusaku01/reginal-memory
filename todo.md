# 最終的なアプリのイメージ

- GoogleMap と Figma の融合（細かい機能まで似せる必要はない）

  - 最背面レイヤーには Map をセットし、その上にツールバーを駆使して作成したオブジェクトをレイヤーとして重ねていくイメージ
  - 最終的にできたレイヤーの重なり物は、保存ボタンで 1 つの画像として保存される
